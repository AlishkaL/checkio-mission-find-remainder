Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Rozwiązanie, które Państwu proponuję jest niezwykle proste. Proszę poczytać i użyć operatora modulo.

Ok, od czego powinienem zacząć?

Proszę poczytać o operatorze modulus i korzystać z niego! 

remainder = dividend % divisor;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik lub całe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Poniżej znajduje się całe rozwiązanie:

function findRemainder(dividend: number, divisor: number): number {

    return dividend % divisor;
}

Jasne, proszę pobrać!

W tym rozwiązaniu używana jest rekurencja. 

function findRemainder(dividend: number, divisor: number): number {

    if (dividend < divisor) return dividend;
    
    return findRemainder(dividend - divisor, divisor);
}

Proszę znaleźć iloraz, pomnożyć przez dzielnik i odjąć od dywidendy. 

function findRemainder(dividend: number, divisor: number): number {

    let quotient = Math.floor(dividend / divisor);
    
    return dividend - (quotient * divisor);
}
