Ok, od czego powinienem zacząć?

Proszę poczytać o operatorze modulo i korzystać z niego! 

remainder = dividend % divisor
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik lub całe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def find_remainder(dividend: int, divisor: int) -> int:

    return dividend % divisor

W tym rozwiązaniu używana jest rekurencja. 

def find_remainder(dividend: int, divisor: int) -> int:

    if dividend < divisor: return dividend
    
    return find_remainder(dividend - divisor, divisor)
